##  Project Description
Movie Explorer is a React web application that allows users to
browse popular movies, search for specific titles, filter by genre,
and view detailed information about each movie.
The app uses the TMDB (The Movie Database) API to fetch
real and up-to-date movie data.

---

##  Features
- Browse popular movies
- Search movies by title
- Filter movies by genre
- View full movie details
- Add a movie via a validated form
- Fully responsive design

---

##  Technologies Used
- React
- React Router DOM
- Bootstrap 5
- TMDB API
- Vite
